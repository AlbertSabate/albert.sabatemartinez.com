### 📋 Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v20 or higher) - [Download](https://nodejs.org/)
- **Bun** (recommended) - [Install](https://bun.sh/)
- **Git** - [Download](https://git-scm.com/)
- **Cloudflare Account** (for deployment) - [Sign up](https://dash.cloudflare.com/sign-up)

### 🚀 Quick Start

#### Step 1: Clone the Repository
```bash
git clone https://github.com/AlbertSabate/cv.git
cd cv
```

#### Step 2: Install Dependencies
```bash
# Using Bun (recommended)
bun install

# Or using npm
npm install
```

#### Step 3: Set Up Environment
```bash
# Copy environment example
cp .env.example .env

# Add your Cloudflare credentials (for deployment)
# CLOUDFLARE_API_TOKEN=your_token_here
```

#### Step 4: Start Development Server
```bash
# Start the dev server
bun run dev

# Open http://localhost:5173
```

### 🧪 Testing & Quality

#### Step 1: Run Type Checking
```bash
bun run typecheck
```

#### Step 2: Run Linting
```bash
# Check for issues
bun run lint

# Auto-fix issues
bun run format
```

#### Step 3: Build for Production
```bash
# Create production build
bun run build

# Test production build locally
bun run preview
```

### 🚢 Deployment

#### Option A: Automatic Deployment (Recommended)

##### Step 1: Set up GitHub Secrets
1. Go to your repo's Settings → Secrets → Actions
2. Add these secrets:
   - `CLOUDFLARE_API_TOKEN`
   - `CLOUDFLARE_ACCOUNT_ID`

##### Step 2: Push to Main Branch
```bash
git add .
git commit -m "feat: update portfolio"
git push origin main
```

##### Step 3: Monitor Deployment
- Check Actions tab for deployment status
- Preview deployments available on PRs

#### Option B: Manual Deployment

##### Step 1: Login to Cloudflare
```bash
wrangler login
```

##### Step 2: Deploy to Production
```bash
bun run deploy
```

##### Step 3: Deploy Preview (Optional)
```bash
wrangler deploy --env preview
```
